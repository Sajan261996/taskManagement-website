# Task Management Application

A full-stack task management application where users can securely manage their personal tasks.

## Features
- User authentication (JWT)
- Create, update, delete tasks
- Mark tasks as completed
- User-specific dashboards
- Protected API routes

## Tech Stack
Frontend:
- HTML, CSS, JavaScript

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Live Demo
Frontend: https://taskmanagemanagement.netlify.app/
Backend: https://taskmanagement-backend-dyfk.onrender.com

All APIs tested using Postman.
